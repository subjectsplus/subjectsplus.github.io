{{ page.title }}

Version 5.0
Developed from the ground up using modern tools: Symfony, API-Platform, Twig, React & MySQL

Advanced Features

Book List

  • Create curated book lists from ISBN numbers
  • Link book entries to your catalog, Google Books or Open API

A-Z Databases List

  • Create lists within guides from official database subscriptions or other resources
  • Overide the title/description for any item on a list

Third-party Integration

  • API: Integrate SubjectsPlus content in websites or applications, providing a seamless research experience
  • Course Connection: Leverage LTI 1.3 technology to connect SubjectsPlus guides with Blackboard Learn courses
  • Authentication: SAML for Single Sign-On (SSO) authentication allows access ot library resources with existing login credentials

Customization

  • Customize any page to match your institution's web presence
  • Add a logo & custom colors from the theme config interface
  • Advance options: edit the provided custom theme to include your own header/footer, or create a new theme

Use Case Examples

Guides

  • Different Content Types: Database lists, book lists, catalog search & librarian profiles
  • WYSIWYG: Add text, links, images, videos & database resources. Organize guide content via drag-and-drop interface, no coding experience necessary.
  • Display guide navigation as tabs or sidebar menu

Resources

  • Manage Resources including A-Z database subscriptions
  • Filter resources by subject area or format

Users

  • Organize users by departments, subject specialists or subjects areas
  • Assign different roles
  • Disaster Map: Locate your staff in case of emergencies
